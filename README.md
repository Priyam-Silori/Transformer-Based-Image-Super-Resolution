# Transformer-Based Image Super Resolution

## Project Overview

This repository contains an implementation and study of a Transformer-based GAN architecture for image super-resolution. The project explores how Vision Transformers and self-attention mechanisms can be used to reconstruct high-resolution images from low-resolution inputs.

The implementation is based on the SRTransGAN research work proposed by Neeraj Baghel, Satish Singh, and Shiv Ram Dubey.

## Objectives

* Understand Transformer-based image restoration
* Explore GAN training for super-resolution
* Analyze self-attention in computer vision
* Study PSNR and SSIM evaluation metrics
* Experiment with modern image enhancement techniques

## Architecture

The model consists of:

* Transformer-based Generator
* Vision Transformer Discriminator
* Adversarial Training Framework
* Self-Attention Mechanisms
